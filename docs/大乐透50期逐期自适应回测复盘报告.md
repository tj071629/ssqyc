# 大乐透50期逐期自适应回测复盘报告

## 1. 回测设定
- 历史窗口：固定 `50` 期。
- 每期输出：程序化生成 `5` 注单式预测。
- 执行方式：先用固定规则跑完整历史，再用逐期在线修订规则跑同一批历史，比较两者差异。
- 自适应逻辑：每预测1期，先识别前区状态并动态分配5注模板，再根据真实开奖号码更新后区特征权重和票型优先级，下一期直接使用已更新后的规则。

- 后区正向增强特征：`neighbor_last(0.01), omission_mid(0.01), repeat_last(0.01), omission_step(0.00)`

## 5. 程序归纳出的修正规则
- 最近1期承接层不再只是固定顺序，而是以“主承接票”形式参与动态竞争；当这一路线持续命中时，会自动保留在高优先级票型里。
- 断区票、逆向跳号票不再机械固定5注同配，而是根据 `extreme / rebound / steady / mid-heavy` 状态动态分配。
- 候选池命中通常高于最终出票命中，因此收口阶段改为“票型竞争 + 动态优先级”，不再固定死顺序。
- 后区不再机械重复押同一 exact pair，而是根据活跃、邻接、遗漏特征的实时表现调整。
