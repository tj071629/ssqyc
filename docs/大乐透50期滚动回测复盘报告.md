# 大乐透50期滚动回测复盘报告

## 1. 回测设定
- 历史窗口：固定 `50` 期。
- 每期输出：程序化生成 `5` 注单式预测。
- 预测流程：前区综合最近1期承接、近10/20/50期热度、遗漏回补、对子骨架；后区综合近5/10期活跃、重号、边码、遗漏。
- 复盘口径：每个窗口只与紧随其后的下一期开奖对比，然后滚动前移1期重新训练。

## 3. 规则修正建议（程序归纳）
- 最近1期承接信号必须保留在主线：滚动样本里，实际开奖平均有较高比例来自上一期边码或重号延续，不能把承接层降为可有可无。
- 前区前15候选池比最终5注更稳定：实际开奖平均有较多号码能进入前15，但收口时容易因为均衡思维过早删掉极端结构位。
- 断区与4码堆积不是尾部噪声：历史中断区和单区高堆积出现频率并不低，必须把它们设计成显式票型。
- 后区不能只押活跃延续：后区前6候选覆盖力较强，但 exact pair 很不稳定，说明应优先保留“活跃+边码”“活跃+回补”结构，而不是重复压同一对。
- 逆向跳号层有必要保留：不少失手来自主线候选过于贴近上一期，缺少1注明确的脱核重组票。
