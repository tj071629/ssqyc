# 大乐透2胆5拖逐期回测复盘报告

## 1. 回测设定
- 历史窗口：固定 `50` 期。
- 每期输出：`前区2胆 + 5拖码 + 后区2码`，等价 `10` 注单式，不含追加金额为 `20元`。
- 复盘口径：前区以胆拖隐含的 `10` 注里最优前区命中为准；后区因固定 `2` 码，所有隐含注共享同一后区结果。
