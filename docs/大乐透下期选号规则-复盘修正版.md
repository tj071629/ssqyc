# 大乐透下期选号规则（复盘修正版·理论增强版）

## 1. 目的
本规则用于下一期大乐透选号，在实战复盘修正的基础上，融合了《构建基于理论框架与统计分析的彩票号码生成与长期研究跟踪方法论》和《基于知识库构建的乐透分析与投注决策方法体系》两份方法论的核心内容，形成"实战经验 + 理论框架 + 统计工具"三位一体的选号体系。

## 2. 理论基石

### 2.1 核心认知
- 根本属性：大乐透开奖号码是真随机产生的，单期预测在数学本质上无效。
- 分析前提：大量历史数据会呈现稳定的统计规律，分析目标是将盲目选择转化为策略性覆盖。

### 2.2 全局均衡三原理
这是统御所有分析的最高指导原则：
- 非对称原理：指标不会长期保持完美对称（如奇偶比不会永远 `3:2`）。
- 非等量原理：短期内各指标状态出现次数不均等，会出现"偏态"。
- 求均衡原理：长期来看，任何指标的实际频率都会趋近理论概率。分析的本质是识别当前偏态，判断其回归均衡的时机与方式。

### 2.3 六大分析维度
从微观到宏观、从静态到动态的结构化分析透镜：
1. 基础分布属性：和值（理论均值 `90`）、极差（理论均值 `24`）、首尾和等中心趋势与离散度指标。
2. 号码分类形态：奇偶（前区第1/3/5位奇数占优）、大小、质合、012路、尾数分布。
3. 号码间拓扑关系：聚集效应（号码倾向于数值上彼此靠近）、连号与间隔、历史关联（重码、边码、跟随号、对称数、相生/相克组合）。
4. 动态演变指标：遗漏分析、冷热分层、多期走势（移动平均线、趋势线）。
5. 结构位置特征：排序定位法（X1-X5各位置候选范围）、区间与断区规律。
6. 全局均衡原理：识别偏态 → 判断回归时机。

## 4. 核心原则
1. 最近1期承接信号上调为前区第一优先级，优先于多期斜连。
2. 历史重组仍保留，但只负责提供骨架，不再主导多票的共同主线。
3. 前区必须保留 `1` 个逆向跳号，但默认不再强求 `2` 个，避免为了防偏移而把主线打散。
4. 低中高区均衡改为软约束，允许任一大区在强断区信号下临时缺失，不能再把"三区都得有号"当成隐性硬过滤。
5. 后区必须分开考虑"活跃延续号、斜连号、回补号"，禁止把连续活跃号简单降权。
6. 若一次购买多张票或套餐票，必须限制同核重复率，不能让多数票都围绕同一组骨架展开。
7. 追热不追冷：以热号和温号为主体，冷号仅在出现明确"解冻"信号时谨慎配置。
8. 宁精毋滥：聚焦趋势最明确、规律性最强的少数核心信号，避免条件过多导致干扰。
9. 断区不是异常态，而是常见结构态；最近 `3~6` 期若断区频发，单票也必须允许按断区结构直接组号。
10. 上一期若出现 `0:1:4`、`4:1:0` 这类极端偏态，下一期必须同时检查"反抽票"，不能只做偏态延续票。
11. 低区回补应优先以小池思维处理（如 `02~05`），不能在没有足够共振证据时过早收口成单点号。
12. 后区活跃延续号若连续 `2` 期未兑现，应主动降权，避免被上一期命中记忆绑架。
13. 若上一期已经完成一次反抽，下一期多票中必须至少保留 `1` 注回稳票，优先结构为 `1:2:2` 或 `2:2:1`。
14. 若后区同时出现"活跃延续号 + 回补号"的双线共振，至少保留 `1` 组主配对票，不得全部拆散到不同票里。
15. 回稳阶段默认按 `2:1:2 / 1:2:2` 双态摆动处理，多票中不得只保留其中一种结构。
16. 低区回补若已从 `02~05` 一类单点小池上移到 `06~08` 一类活跃簇，多票中必须至少保留 `1` 注低区双点票。
17. 后区活跃簇未断时，`双活跃配对` 必须与 `活跃 + 回补` 同时保留，不能只押后一种。
18. 若上一期同时开出 `3` 个中高骨架号，下一期必须执行"三重号限幅"：主复式默认只保留其中 `2` 个，单式多数票只保留 `1~2` 个，三号同留最多只允许 `1` 注进攻票。
19. 区带结构复现不等于号码主线复现；当 `2:1:2 / 1:2:2` 再次出现时，必须先判断是结构延续还是旧号码延续，不能默认两者同步成立。
20. 多票中必须保留 `1~2` 注"脱核重组票"：结构可以延续，但号码内容要主动与上一轮旧骨架拉开，转向冷温重组号。
21. 若最近1期后区为双活跃或上一轮主压为双活跃，下一期必须同时保留 `1` 组"活跃回访 + 回补"配对，不得继续只押双活跃。若后区某活跃号已连续 `5` 期仍未断开，下一期必须强制降权为备选，不得继续无条件锁定。
22. 若最近 `6` 期内 `1:4` 已出现 `2` 次及以上，或刚出现 `5:0 / 0:5` 这类奇偶极端，应结合最近 `2~3` 期走势检查是否存在镜像反抽信号；若信号成立，`4:1` 或 `1:4` 仅作为可放行分支之一，多票可保留 `0~1` 注，不得机械固定。
23. 若上一期前区已出现 `4:1:0` 或低区四点成簇，应结合最近 `2~3` 期低区活跃度、高区走弱与断区/准断区信号决定是否放行低区四点票；若放行，多票可保留 `0~1` 注，允许高区为 `0` 或仅保留 `1` 个补位号，不得机械固定。
24. 上一期后区 exact pair 若已直接开出，下一期该 exact pair 在手工多票中最多只允许保留 `1` 次；其余票必须转向同逻辑替代配对，不得在主复式与多张单式里重复重压。
25. 若最近1期出现重号延续（同号连续两期），边码规律可能暂时让位于重号模式，此时应优先锁定重号而非仅追踪边码。
26. 高区成簇记忆号（如 `25/27/29/34` 一类）进入候选池后，必须至少集中 `2` 个到同一张主票，不得分散到不同票里被"均衡思维"稀释。
27. 当连续 `7` 期以上无断区时，必须至少准备 `1` 张断区票型（如 `0:1:4`、`0:2:3` 或 `1:0:4`），不能只停留在"防断区"认知层。
28. 后区冷号（`3` 期以上未出）与活跃断开号的组合（如 `02/06`）必须进入候选池，不能只押活跃延续或回补单一逻辑。

说明：三重号限幅是组合暴露控制，不是否定三重号继续开出的可能。

## 5. 前区选号框架
前区 5 码或 7 码候选池，按 5 层构成：

### 5.1 第一层：最近1期承接号（1~2 个，优先层）
来源：
- 上一期直接重号（遗传号）；
- 上一期前区号码的 `±1` 近邻承接（边码/斜连码）；
- 上一期密集段的外扩或补口位。

要求：
- 若上期出现 2 连、3 连或明显密集段，下一期必须优先保留 `1~2` 个承接号；
- 这层可以直接覆盖到低区或中区，不得因为"区带不均衡"被先删掉。

### 5.2 第二层：历史重组骨架号（1~2 个）
来源：
- 最近 50 期重复较多的对子、三数组合；
- 最近 15 期仍活跃的共现号码；
- 黑洞数字（理论高频号码）中与近期热号的交集。

前区黑洞数字参考（21个）：`01, 02, 03, 04, 08, 09, 10, 11, 16, 17, 18, 19, 20, 25, 26, 27, 28, 32, 33, 34, 35`。

要求：
- 只保留 `1~2` 个，作为结构稳定器；
- 若最近1期承接已经很强，历史骨架不得反客为主；
- 优先从"热号 ∩ 黑洞数字"交集中选取。

### 5.3 第三层：辅助斜连号（0~1 个）
来源：
- 最近 1~3 期 `±1` 近邻斜连；
- 相邻期的延伸关系；
- 对称数辅助：前区以 `18` 为对称轴，已选号码的对称数（`对称数 = 36 - 原号码`）可作为备选参考。

要求：
- 只用于补位，不得超过 `1` 个主代表；
- 不得再让整张票由同一条近邻斜连带着走。

### 5.4 跨多期斜连现象（辅助观测层）
历史数据显示，每期开奖号码中通常有 `1~3` 个号码可以对应到跨多期的等差递进模式，这不是偶发现象而是常态。

识别方法：
- 取最近 `5~10` 期的前区号码，逐号检查是否存在公差为 `1~4` 的等差数列（至少 `3` 期连续递进）；
- 例如：第N-2期出现 `08`，第N-1期出现 `10`，则公差 `+2` 的斜连指向本期 `12`；
- 同一期可能存在多条斜连线索，需要筛选而非全部采纳。

应用规则：
- 每期从跨多期斜连中提取 `1~2` 个候选号码，作为独立的观测信号；
- 当斜连方向与第一层承接号指向一致时，视为"双线共振"，该号码优先级提升；
- 当斜连方向与承接号指向矛盾时，以承接号为主，斜连号降为备选；
- 禁止 `3` 条以上斜连线索同时主导一张票——多条斜连应分散到不同票中；
- 公差越小（`+1`/`-1`）、连续期数越长的斜连，可信度越高。

后区同理：后区号码域小（`01~12`），等差模式更容易识别，可取最近 `8~12` 期观察。

### 5.4 第四层：逆向跳号（1 个，必须有）
来源：
- 模型前列之外、但处于合理区间补位位置的号码；
- 最近1期完全未覆盖、且能修正结构单一问题的号码；
- 边沿数字（`01-04` 或 `32-35`）中近期遗漏较深但处于合理回补窗口的号码。

要求：
- 必须保留 `1` 个；
- 作用是防止收口时把真实开奖号码中的偏移位删掉；
- 若已出现明显强承接，不应再为了凑第 `2` 个跳号而破坏主线。

### 5.5 第五层：结构平衡号（可选 0~2 个）
用于修正：
- 奇偶比例（目标 `2:3` 或 `3:2`，注意前区第1/3/5位奇数理论占优）；
- 过密连号（利用位差PD值检查，小PD值 `1~7` 为高频，奇数PD更优）；
- 单一数段过重；
- 5 码或 7 码的最终票面观感。

### 5.6 历史组合克隆现象（辅助观测层）
历史数据中存在一种"组合克隆"现象：某期开奖号码并非全新组合，而是最近若干期号码的部分重组拼合。

识别方法：
- 取最近 `3~5` 期的全部前区号码（共 `15~25` 个），去重后形成"近期活跃号码池"；
- 观察该池中是否存在 `3` 个及以上号码在本期同时开出的情况——若频繁出现，说明当前处于"克隆活跃期"。

应用规则：
- 当判断为克隆活跃期时，可从近 `3~5` 期号码池中直接提取 `2~3` 个号码作为候选，与第一层承接号形成互补；
- 克隆号不等于照搬：允许从多期中各取部分号码重新拼合，禁止整组照搬某一期；
- 克隆号与第一层承接号可能重叠，重叠部分视为"双重确认"信号，优先级提升；
- 当近期号码池覆盖度很高（如 `15~25` 个去重号已覆盖大半号码域）时，克隆信号的区分度下降，此时应降低该层权重。

后区同理：取最近 `5~8` 期后区号码（共 `10~16` 个），观察重组拼合频率，辅助后区候选池筛选。

## 6. 前区约束修订与量化校验
最终前区应优先满足：
- 低区（01~12）优先有 `1` 个，但在强断区信号下允许没有；
- 中区（13~24）优先有 `1` 个，但在强断区信号下允许没有；
- 高区（25~35）优先有 `1` 个，但在强断区信号下允许没有；
- 奇偶尽量保持 `2:3` 或 `3:2`；
- 允许出现 `1` 组 2 连或 3 连；若最近1期强承接明显，出现更长密集段时不得因主观惧怕连号而全部删掉；
- 不允许 `4` 个以上号码全部来自同一条斜连逻辑；
- 不允许整张票原样照搬历史某一期号码。

### 6.1 断区识别与放行规则
断区定义：低区（`01~12`）、中区（`13~24`）、高区（`25~35`）三大区中，任意一个区在当期前区 `5` 码里出现 `0` 个号码。

当满足以下任意 `2` 条时，视为"强断区信号"：
- 最近 `6` 期内出现 `2` 次及以上断区；
- 上一期三区分布极端偏斜，如 `4:1:0`、`0:1:4`、`3:2:0`；
- 上一期存在连续密集段后，下一期候选池明显向另一侧区间迁移；
- 排序定位出现异常上跳或下沉，如 X1 明显抬高到 `13+`，或 X5 明显压低到 `24-`；
- 前区位差出现 `10+` 的大跨距，并同时伴随一侧聚集。

放行规则：
- 一旦判定为强断区信号，允许单票直接采用"某一区为 0 个"的结构；
- 断区票优先控制为 `1区缺失 + 其余两区分配 1:4 或 2:3`；
- 若只买单票，不得因为"观感不均衡"就否掉断区结构；
- 若买多票，至少保留 `1` 注断区票，且不得所有票都按均衡票处理。
- 若最近 `3~6` 期已出现不同大区轮换断区，下一期必须显式检查"另一大区继续断区"而非仅检查上一期断区回补；多票时至少保留 `1` 注轮换断区票。

### 6.2 极端结构后的反抽规则
- 若上一期前区为 `0:1:4`、`4:1:0`、`0:2:3` 这类极端偏态，下一期默认同时准备 `1` 注反抽票；
- 反抽票优先结构为 `2:1:2`、`2:2:1` 或 `1:2:2`，重点防"两端成簇 + 中轴单点"；
- 若低区判断为回补，不得过早把候选池收成单点号，应至少保留 `2~3` 个低区备选再压缩；
- 多票方案中，偏态延续票与反抽票必须并存，不能全部押在同一条延续主线上。

### 6.3 反抽后的回稳规则
- 若上一期前区已经走出一次明显反抽（如 `2:1:2`），下一期多票中必须显式保留 `1` 注回稳票；
- 回稳票不得理解为固定单结构；当前若存在摆动信号，多票时应同时检查 `1:2:2` 与 `2:1:2`，`2:2:1` 作为扩展分支；
- 回稳票重点防"低区单点 + 中高双支撑"与"低区双点 + 中轴单点 + 高区双支撑"两类主结构；
- 回稳票中，旧骨架若已连续 `2` 期未兑现，应主动降权，避免继续由 `10/18/29` 一类老主线占据核心位置；
- 回稳不等于均衡平均分配，而是承认结构已从极端/反抽切换到更稳定的温冷重组态。
- 若低区回补已呈现上移迹象，不得继续只按 `02~05` 单点小池收口，至少保留 `2` 个低区活跃候选并在多票中显式安排 `1` 注低区双点票。
- 若上一期同时开出 `3` 个中高骨架号（如 `22/29/34`），则回稳阶段不得再把这 `3` 个整体当作统一主干；主复式默认只保留其中 `2` 个，第三个仅作为单式进攻分支使用。
- 若 `2:1:2 / 1:2:2` 的结构再次出现，必须先判断本期是"结构复现"还是"号码复现"；若缺乏强共振证据，默认按"结构复现 + 脱核重组"处理。
- 当判断为"结构复现 + 脱核重组"时，前区候选池必须补入 `2~3` 个冷温重组号，主复式不得继续以旧三核心为轴。
- 回稳阶段的"结构回归"不等于旧骨架整体回归；若旧中高骨架已连续重压但未兑现，即使区带重新回到 `1:2:2 / 2:1:2`，也应优先按"结构延续 + 内容换血"处理，不得机械把 `13/22/29/34` 一类旧轴重新收回主复式核心。
- 若高区已有 `2` 个相邻或近邻号进入主池（如 `26/27`），必须显式检查其补口位（如 `25` 或 `28`）是否应进入至少 `1` 注；高区补口位的职责不是追高，而是防止高区密集簇只抓到轴心和延展、漏掉成簇闭环。

### 6.4 奇偶极端镜像 / 低区极端放行规则
- 若最近 `6` 期内 `1:4` 已出现 `2` 次及以上，或刚出现 `5:0 / 0:5` 极端奇偶，下一期应先检查是否存在奇偶镜像反抽信号；若信号成立，多票中可放行 `0~1` 注 `4:1` 或 `1:4` 镜像票；
- 奇偶镜像票的作用是防"上一轮偶重连开后，下一轮奇重反抽"，不得把奇偶比长期固定在 `2:3 / 3:2` 的舒适区，也不得把镜像票机械写成每期必留；
- 若上一期前区已出现 `4:1:0` 的低区极端结构，应结合低区活跃簇、高区走弱与断区/准断区信号判断是否放行低区四点票；若放行，多票可保留 `0~1` 注，允许结构为 `4:1:0` 或 `4:0:1`；
- 低区四点票不等于机械照搬上一期号码，应以边口活跃号、小池回访号和 `1` 个中轴补位号重新组合，避免只复制 `03/05/07/09/18`。

### 6.5 量化校验指标（组合完成后必查）
| 校验项 | 参考标准 | 说明 |
| :--- | :--- | :--- |
| 和值（SUM） | 理论均值 `90`，高概率区间 `70~110` | 大幅偏离需审视原因 |
| 极差（JC） | 理论均值 `24`，高频段 `20~31` | 控制号码跨度与分布密集度 |
| 位差（PD） | 理论均值 `6`，小PD值 `1~7` 高频 | 相邻号差值，奇数PD更优 |
| 奇偶比 | 常规 `2:3 / 3:2`，极端摆动期可放行 `1:4 / 4:1` | 最近 `1:4` 高频时应显式检查是否需放行 `4:1 / 1:4` 镜像分支 |
| 大小比 | `2:3` 或 `3:2` | 以中位数 `18` 分区 |
| AC值 | 通常 `≥4` | 衡量号码间关系的复杂度 |
| 聚集效应 | 应有 `1~2` 个小密集区 | 号码倾向于数值上彼此靠近 |
| 断区态 | 最近断区频发时允许 `1区为0` | 不得用均衡偏见提前过滤 |

### 6.6 冷热分层量化标准
| 分层 | 遗漏期数 | 配置建议 |
| :--- | :--- | :--- |
| 热号 | ≤5 期 | 主体，选 `2~3` 个 |
| 温号 | 6~9 期 | 辅助，选 `1~2` 个 |
| 冷号 | ≥10 期 | 防守，选 `0~1` 个（需有解冻信号） |

冷号"解冻"信号：长期遗漏后首次开出，未来 `2~3` 期可重点关注。

## 7. 后区选号框架
### 7.1 后区候选池必须分三类
- 活跃延续号：最近 3~5 期反复出现、仍在高活跃区的号码；
- 斜连号：最近1期 `±1` 或短期承接得到的号码；
- 回补号：短期沉寂后处在合理回补窗口的号码。

后区黑洞数字参考（6个）：`01, 02, 03, 10, 11, 12`。
后区对称轴为 `6.5`，对称数 = `13 - 原号码`。

### 7.2 后区 2 码单式规则
- `1` 个优先从"活跃延续号 / 历史配对池"中选；
- `1` 个从"斜连号 / 回补号"中选；
- 若两个号都只是同一逻辑来源推出来的，必须替换掉 `1` 个；
- 禁止因为某个后区号刚开过就自动降级，尤其不能错杀连续活跃号。
- 若某个活跃后区号已连续 `2` 期作为主压仍未兑现，下一期默认降为备选，不得继续无条件锁定。
- 若同时识别到"活跃延续号 + 回补号"的强配对关系，至少保留 `1` 组主配对票，不能把两个号完全拆散到不同票里。
- 若最近 `2~3` 期后区活跃号之间呈交替兑现或簇状活跃，多票中还必须额外保留 `1` 组双活跃配对票。
- 若最近1期后区已开出双活跃，或上一轮主压集中在双活跃上，下一期还必须显式保留 `1` 组"活跃回访 + 回补"配对（如 `01+12` 一类），不得继续只押双活跃。
- 若上一期某组后区 exact pair 已直接开出（如 `01+12`），下一期该 exact pair 在手工多票中最多只允许保留 `1` 次，且不得同时进入主复式与多张单式；其余票必须转向同逻辑替代配对。
- 若后区两个候选号分别已进入不同票面，且它们组合后正好构成当前最强的"活跃回访 + 回补"替代配对，则至少要有 `1` 注把这两个号显式配成一组主票；禁止只做散点覆盖、不做主配对落地。

### 7.3 后区复式规则
- 若预算允许，优先 `3` 码后区；
- 后区 3 码建议结构：`1个活跃延续号 + 1个斜连号 + 1个回补或历史配对号`；
- 若活跃延续号本身兼具历史配对优势，可用 `1活跃 + 1斜连 + 1波动补位` 的结构。

## 8. 多票与套餐票规则
### 8.1 多票分散原则
- 若一次购买 6 注单式或 2 张套餐票，不得让超过一半的票都围绕同一对主骨架号展开；
- 同一组三核心号码在多票中重复出现时，应控制在总票数的一半以内；
- 至少保留 `1~2` 注明显偏最近1期承接的票，不能全部做成历史骨架票。
- 若上一期同时开出 `3` 个中高骨架号，则下一期 8 注单式中，超过半数的票不得同时保留这 `3` 个号；
- 三重号同留票最多 `1` 注，并且必须明确标记为进攻票，不得让主复式承担三重号集体延续风险。
- 若当前判断为"结构复现"而非"号码复现"，则多票中必须保留 `1~2` 注脱核重组票，不得所有票都围绕上一轮旧骨架展开。
- 若最近奇偶已进入极端摆动阶段，多票中应显式检查 `4:1` 或 `1:4` 镜像票是否需要放行；若放行，可保留 `0~1` 注，不得机械固定。
- 同一组后区 exact pair 在手工多票中最多出现 `1` 次；若主复式已使用该对，单式中不得再批量重复。

### 8.2 套餐票实操原则
- 套餐内的复式票负责守主线，但单式票必须承担分散任务；
- 复式与单式不能只是同一套号的小改款；
- 若复式走"历史骨架 + 承接"，则单式里必须有部分票转向"承接 + 跳号"或"承接 + 回补"。

## 9. 推荐的实际执行步骤（四阶段闭环）

### 阶段一：数据准备与态势感知
1. 导入最新开奖数据，更新遗漏表、冷热分层、各指标偏离度。
2. 计算当前和值、极差、奇偶比、大小比等结构参数，与理论值对比，识别偏态。
3. 检查均衡三原理：哪些指标处于极端偏态，是否接近反转阈值（遗漏反转率 > 75%）。
4. 统计最近 `6` 期三区分布，记录断区次数、断的是哪一区，以及 X1/X5 是否发生异常跳位。
5. 若上一期为极端偏态结构，提前准备一组"反抽票"候选框架，避免只沿上一期主线延续。
6. 若上一期已经完成一次反抽，额外准备 `1` 组"回稳票"候选框架，避免继续沿反抽主线惯性延伸。
7. 若最近 `2~3` 期在 `2:1:2 / 1:2:2` 间摆动，标记为"双态回稳期"，多票时不得只保留一种区带结构。
8. 检查上一期是否同时开出 `3` 个中高骨架号；若是，提前标记本期需执行"三重号限幅"，禁止默认整组沿用。
9. 若本期只是区带结构再次出现，但上一轮旧骨架已被连续重压，则标记为"结构复现期"，提前准备 `1~2` 组脱核重组模板。
10. 额外检查最近 `6` 期奇偶是否已出现多次 `1:4` 或 `5:0 / 0:5` 极端，并同步记录上一期后区 exact pair 是否刚开出，提前标记"奇偶镜像分支是否放行"和"后区去重"。

### 阶段二：候选池生成（五层选号）
1. 先看最近1期前区是否出现重号延续、密集段、`±1` 外扩。
2. 从最近1期承接池里先锁 `1~2` 个号码。
3. 再从最近 50 期对子、三组里补 `1~2` 个历史骨架号（优先"热号 ∩ 黑洞数字"交集）。
4. 从斜连池里只选 `0~1` 个代表号（可参考对称数辅助）。
5. 强制补入 `1` 个逆向跳号（关注边沿数字 `01-04` / `32-35`）。
6. 检查选号密度 D 值：前区 `D = 候选池数量 / 5 ≤ 4`，后区 `D = 候选池数量 / 2 ≤ 3`。

### 阶段三：组合优化与校验
7. 用量化校验指标（和值 `70~110`、极差 `20~31`、AC值 `≥4`）过滤低概率组合。
8. 先判断是否触发强断区信号；若触发，必须显式检查断区票，而不是默认筛掉。
9. 若上一期为极端偏态，必须显式检查至少 `1` 注反抽票，不能只保留偏态延续票。
10. 若上一期已经完成反抽，必须显式检查至少 `1` 注回稳票，不能继续只保留反抽延续票。
11. 若低区回补已呈现上移或成簇迹象，必须显式检查至少 `1` 注低区双点票，不能只留单点回补票。
12. 若上一期存在 `3` 个同开中高骨架号，必须检查主复式是否已限幅为其中 `2` 个，且三号同留票是否被压缩到最多 `1` 注。
13. 若判断为"结构复现"而非"号码复现"，必须检查主复式是否已主动降旧核，并保留了 `1~2` 注脱核重组票。
14. 若最近奇偶已进入极端摆动，必须显式检查是否需要放行 `4:1` 或 `1:4` 镜像票；若放行，多票可保留 `0~1` 注，不能机械固定。
15. 若上一期已出现 `4:1:0` 或低区四点极端，必须显式检查低区四点票是否具备放行条件；若放行，多票可保留 `0~1` 注，并允许高区为 `0` 或 `1` 的压缩结构。
16. 检查奇偶、区带和密集度——注意这些是修正项，不是先验硬过滤器。
17. 检查聚集效应：组合内应自然形成 `1~2` 个小密集区，避免过度分散或全部紧邻。
18. 后区先列出"活跃延续 / 斜连 / 回补"三类候选；若活跃号连续两期未兑现，需主动降权后再压缩到 `2` 码或 `3` 码。
19. 若后区同时看到了活跃延续号与回补号的强配对关系，至少保留 `1` 组主配对票，不得全部拆散。
20. 若后区活跃号处于簇状交替阶段，至少保留 `1` 组双活跃配对票。
21. 若最近1期或上一轮主压为双活跃，必须显式检查 `1` 组"活跃回访 + 回补"配对，不能只保留双活跃延续；同时检查上一期 exact pair 是否已去重。
22. 若要买多票，做一次同核重复率检查，并确认是否同时保留了断区结构票、反抽票、回稳票、低区双点票、三重号限幅、脱核重组票，以及是否按走势检查了奇偶镜像/低区极端放行分支。

### 阶段四：复盘与迭代
23. 开奖后执行复盘（见第 13 节），将结果录入数据库，驱动下一轮分析。

## 10. 明确禁忌
- 禁止把"高区至少 1 个"继续当成硬性过滤条件。
- 禁止在断区高发阶段继续默认"低中高至少各留 1 个"。
- 禁止上一期刚出现极端偏态，下一期就把所有票都做成同方向延续票。
- 禁止上一期已经完成反抽，下一期主复式仍然被旧骨架绑住，不做回稳票。
- 禁止把回稳阶段理解成固定 `1:2:2`，忽略 `2:1:2` 的摆动态。
- 禁止上一期同时开出 `3` 个中高骨架号，下一期仍把这 `3` 个继续当成整套票统一主干。
- 禁止把 `2:1:2 / 1:2:2` 的结构复现，直接等同于上一轮旧号码主线继续有效。
- 禁止最近 `1:4` 已多次出现、且走势已给出镜像反抽信号时，下一期仍然完全不检查 `4:1 / 1:4` 奇偶镜像分支。
- 禁止上一期已走出 `4:1:0` 低区极端、且低区续强/高区走弱信号成立时，下一期仍然用"高区至少 1 个"的惯性思维把低区四点票提前过滤。
- 禁止只凭多期斜连直接定整张票。
- 禁止看到历史某期顺眼就整组照搬。
- 禁止后区只因"上次最强"或"刚开过"就持续压死或直接删掉。
- 禁止低区回补已明显上移成簇，仍然只按 `02~05` 单点小池机械收口。
- 禁止后区活跃簇未断，就只保留"活跃 + 回补"而不留双活跃票。
- 禁止上一轮双活跃刚兑现或被连续主压，下一期仍然完全不留"活跃回访 + 回补"配对。
- 禁止把上一期已经开出的后区 exact pair，在主复式和多张单式里继续批量重压。
- 禁止多张票看起来很多，实际上都押在同一组三核心号上。

## 11. 复式与胆拖建议
### 11.1 若买 7+2
建议结构：
- 前区：`2个最近1期承接 + 1~2个历史骨架 + 1个辅助斜连 + 1个逆向跳号 + 1个平衡号`
- 后区：优先 `1活跃延续 + 1斜连或回补`。

### 11.2 若买 7+3
这是更稳的复式结构：
- 前区仍按上面规则；
- 后区建议覆盖 `1活跃延续 + 1斜连 + 1回补`，避免再次把后区压死。

### 11.3 胆拖投注策略
胆拖是将"高确定性号码"与"扩大覆盖号码"结合的高效方式：
- 胆码（每注必含）：从第一层承接号和第二层骨架号中，精选 `2~3` 个信心最高的号码。
- 拖码（用于补足）：将候选池中剩余号码作为拖码。
- 注数计算：`注数 = C(拖码数, 需补足号码数) × 后区组合数`。
- 胆码不宜超过 `3` 个，否则容错率极低；胆码必须从热号区、邻号中选取。

### 11.4 缩水与过滤
当复式或胆拖生成注数过多时，使用条件过滤缩水：
- 过滤条件：和值范围、极差范围、奇偶比、大小比、连号组数、AC值等。
- 旋转矩阵：实现"中N保M"的科学组合优化（如"中5保4"矩阵）。
- 容错设置：允许 `0~2` 个过滤条件出错，避免因个别条件误判而错失全部中奖组合。

- 多票投注时若前区同核重复率过高，即使中了部分结构，也容易集中落在同一低奖级，因此分散主线是必要条件。

## 13. 复盘口径
每次开奖后必须复盘二十二件事：
1. 最近1期承接信号是否被候选池正确保留；
2. 历史骨架是否压得过重，导致区带切换时整体失真；
3. 是否出现了断区或准断区结构，而候选池/过滤规则没有放行；
4. 后区错在候选池没看到，还是错在把活跃延续号删掉了；
5. 多票之间是否只是换壳，没有真正分散主线；
6. 量化校验指标（和值、极差、奇偶比等）的过滤条件是否合理——开奖号码的实际参数是否落在设定范围内，若经常偏离需调整参数；
7. 冷热分层判断是否准确——热号是否持续？温号转换判断是否正确？
8. 上一期极端偏态后，是否错误地只做了延续票，而没有保留反抽票；
9. 低区回补与后区活跃号降权，是否在执行时被过早收口或被旧命中记忆绑架；
10. 上一期若已完成反抽，是否把回稳误判成单一结构，漏掉了 `2:1:2 / 1:2:2` 的摆动态；
11. 低区回补是否已升级为双点簇，却仍按单点小池思路收口；
12. 后区是否处于双活跃阶段，却过早只保留了"活跃 + 回补"；
13. 后区是否看到了"活跃 + 回补"的主配对关系，却在实战中被拆散掉；
14. 多票之间是否虽然分散了中高主干，却没有给低区双点簇单独留出成体系的票；
15. 上一期若同时开出 `3` 个中高骨架号，是否仍把它们继续当成统一主干，没有执行限幅；
16. 主复式是否错误承担了三重号集体延续，而不是只保留其中 `2` 个核心号；
17. 是否把 `2:1:2 / 1:2:2` 的结构复现误判成上一轮旧号码主线复现；
18. 是否漏掉了应保留的 `1~2` 注脱核重组票；
19. 后区是否被上一轮双活跃结论绑住，没有同步保留 `01/12` 一类"活跃回访 + 回补"配对；
20. 最近 `1:4` 已多次出现、且走势给出镜像反抽信号时，是否仍然漏掉了应放行的 `4:1 / 1:4` 奇偶镜像分支；
21. 上一期若已走出 `4:1:0` 低区极端，且低区续强/高区走弱信号成立时，是否没有放行低区四点票或高区断区/准断区票；
22. 是否把上一期后区 exact pair 重复压成主轴，而没有拆成同逻辑替代配对；

## 15. 资金管理与风控纪律

### 15.1 资金分配"三分法"
将闲余资金严格按以下比例划分：
- 储蓄备用金（1/3）：绝对不可用于购彩。
- 风险购彩金（1/3）：购彩资金的绝对上限。
- 机动准备金（1/3）：仅在发现绝佳机会时谨慎追加。

### 15.2 止损止盈纪律
- 单期预算上限：不超过购彩金的 `10%`。
- 单期止损：亏损达该期预算 `50%` 时，强制停止追加。
- 连续止损：连续 `5` 期未中奖，强制暂停，退回复盘分析。
- 止盈：中得三等奖及以上后，大幅降低后续投注额度或暂停。

## 16. 一句话版本
下期大乐透的新规则不是"历史骨架 + 斜连一路推到底"，而是：
- 先看最近1期承接，
- 再补历史骨架，
- 留 1 个跳号防删错，
- 后区同时保留活跃延续与回补，
- 多票必须真正分散，而不是同押不同写法。
