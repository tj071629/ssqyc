## 提升空间判断

1. **全量最优**：「融合Top50并集3码」59/8538 = **0.691%**，约为随机的 **1.31x**、理论的 **1.15x**。
2. **绝对提升**：相对随机 3 码约 **+0.164 个百分点**（14 次 / 8538 期）。
3. **统计意义**：最优策略 Wilson 下界 **0.536%**，随机基线 **0.394%**；最优略高，但优势幅度仍很小。
4. **天花板**：每期只有 120 种三元组，随机期望 0.6%；最好的规则也只是 **多中几十次/8000+期**，不具备「可量产」的跃升。
5. **可做的不是提命中率，而是降方差**：
   - 用 **融合Top30并集3码** 或 **热2+遗漏1** 替代纯热Top3，全量略稳；
   - **避开组三/豹子期**无法通过选号解决（结构占 ~28%）；
   - 同等 12 元可改 **2 组 3码**（12注）分散不同三元组，提高「至少中一次」概率，而非单池提率。

## 实操建议（在 12 元预算内）

| 目标 | 建议 |
| --- | --- |
| 单池求稳 | **热2+遗漏1** 或 **融合Top30并集3码** |
| 略追全量最优 | 采用搜索排行 Top1 规则，但预期仅 +0.01~0.02pp |
