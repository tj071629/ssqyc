# 双色球下期选号规则（复盘修正版）

## 1. 目的
本规则用于下一期双色球选号，基于最近一次复式实战复盘重写，重点解决"红球方向有一定命中但收口错误、蓝球单压过死"的问题。

## 2. 上次复盘结论
- 上次红球并非完全跑偏，实际开奖号码中有多个号码原本就在模型较高位。
- 最终问题不在候选号池完全错误，而在 `7红` 收口时删错了该保留的号码。
- 蓝球只压 1 个且过度依赖斜连，导致蓝球命中率偏低。
- 结论：双色球模型可以继续沿用，但必须重写"收口规则"。

- 这次失手的核心不在单个号码，而在于过早把票面又拉回高区热段，低估了 `12/13/18` 一类中段承接与压缩走势。

## 3. 新版核心原则
1. 保留"多期斜连 + 历史重组"双主线。
2. 红球最终 `7` 码必须强制加入 1 个"即时跳点号"。
3. 蓝球不再只靠斜连定 1 码，必须叠加近期波动和遗漏回补。
4. 历史重组用于搭骨架，不用于机械复刻旧号码。
5. 最终收口优先保证覆盖不同逻辑来源，而不是只取模型高分前 7。
6. 当高位骨架已经较重时，最后一个补位优先考虑"结构平衡号/区带回摆号"，防止中段真实落点被删掉。
7. 双色球也要考虑"准断区 / 断区"，但优先级低于大乐透；实战上重点防"某一区只开 1 个"的准断区和上一期偏斜后的区带回摆。
8. 若最近连续两期出现中区偏强（如 `2:3:1`、`1:3:2`），下一期应优先保留 `2~3` 个中区号，不得想当然地把票面重心重新推回高区。
9. 当高区本期只预计留 `1` 个时，优先考虑边沿跳点号，而不是继续追压近期最热的高区骨架号。
10. 若上一期存在 `2~3` 个强重号候选，多注方案里必须至少保留 `1` 注重号集中票，不能把所有重号平均拆散。
11. 若最近连续两期为 `2:3:1` 且中区已明显强势，多注方案里仍必须保留 `1` 注低区增强票，优先结构为 `3:1:2`。
12. 蓝球若已采用分散策略并出现深冷命中迹象，后续复盘应优先检查红球结构问题，不得机械把责任继续归到蓝球单压上。
13. 若最近两期区带结构呈镜像摆动（如 `2:3:1` 后接 `3:1:2`），下一期多注时必须保留 `1` 注 `2:2:2` 均衡回摆票。
14. 上一期复盘结论若明显指向"重号集中 / 低区增强 / 高区边沿"，下一期不得让同一结论继续主导全部票，候选池前端必须额外保留 `2~3` 个均衡重组号。
15. 高区边沿高跳刚兑现后，不得全面压掉 `28/29` 一类热高回切；低区活跃带若已上移到 `03/06/08` 或进一步抬升到 `06/10` 一类偶低带，必须同步保留对应平滑低位号。
16. 若上一期走出 `2:1:3` 且高区由 `24/25/32` 一类新高点主导，下一期不得默认继续三高续强；多注时必须保留 `1` 注 `2:3:1` 回切票，高区只留 `1` 个热高回切位。
17. 当低区承接从 `02/06` 转向 `06/10` 一类偶低上移时，候选池前端不得只保留奇低邻补号，还要同步保留 `10` 一类顺势抬升号。
18. 若最近连续 `2` 期实际开奖奇偶比明显偶重或奇重（如 `1:5`、`5:1`），奇偶约束降为软约束；多注方案里至少保留 `1` 注 `2:5` 或 `5:2` 的偏态票。
19. 蓝球处于分散期时，除了"活跃号 / 邻号 / 冷补号"，还要显式检查 `3~6` 期内的活跃回访号是否应升为最终单压。
20. 若上一期区带走出 `2:3:1` 或中区偏强结构，下一期多注方案里除了保留中区骨架票，还必须额外设计 `1` 注"高区突破票"（高区放行 `3~4` 个号），防止高区大爆发被整体漏掉；高区突破票的高区来源优先考虑 `27/29/31/33` 的全段四点或三点组合。
21. 若前一期走出极端偶重（`1:5` 或 `2:5`），下一期多注方案里**不得**再设置偶重延续票，应替换为 `1` 注重奇票（奇偶比不低于 `5:2`）；连续两期偶重后奇重反弹的概率远高于三连偶重。
22. **主动设计"集中打穿注"**：在多注方案中明确指定1~2注为"主线集中票"，把模型最看好的3~4个核心号强行捆在同一注里，不拆分到不同注中。分散覆盖与集中打穿并行，以追求"这一注打穿"而非"每注都沾边"。
23. **高区归零后弱回补识别**：当上一期高区归零或≤1个号时，本期回补可能是"弱回补"（仅1个号）而非"强回补"（≥2个号）；多注应保留1~2注"高区单点"结构，而非全部预设高区双点。
24. **蓝球中段活跃带启动信号**：当小号活跃带（≤07）连续出现≥3期后，中段号（08~12）可能以"新活跃号"形式出现（如10），而非仅作为"冷补号"；蓝球候选应覆盖中段全带（08~09~10~11~12），至少1~2注分配给中段活跃号。
25. **邻补号扩散速度**：新出现的邻补号在下一期的扩散速度可能快于预期，应在至少2~3注中覆盖，不能仅作为边缘备选。

## 4. 红球选号框架
红球候选池分 5 层：

### 4.1 第一层：历史重组骨架号（2~3 个）
来源：
- 最近 50 期重复较多的对子、三数组合；
- 近 15 期内仍活跃的共现组合。

要求：
- 至少保留 2 个；
- 用于保证整组号码"像历史开奖结构"。

### 4.2 第二层：近邻斜连号（1~2 个）
来源：
- 最近 1~3 期红球 `±1` 承接；
- 相邻期差 1 的延伸关系。

要求：
- 不宜过多；
- 只用于补强，不替代骨架号。

### 4.3 第三层：多期斜连号（1 个）
来源：
- 固定跨距、固定差值的等差递进线。

要求：
- 最多留 1 个代表号；
- 防止整组号码过度线性化。

### 4.4 第四层：即时跳点号（1 个，必须有）
来源：
- 模型前 8~15 名之间，但不是当前骨架或斜连主号；
- 用于防止最终收口时把真实开奖号码中的"跳点"删掉。

这是新版最重要的改动项。

### 4.5 第五层：结构平衡号 / 区带回摆号（可选 0~1 个）
来源：
- 最近 5~10 期内在中区或低区反复活跃，但在当前排序中容易被高位骨架挤掉的号码；
- 最近 1~2 期区带偏斜后，下一期可能出现的中段回摆或低位补口号。

要求：
- 只补 `1` 个；
- 主要用于修正低中高区分布和中段承接，不与即时跳点号重复；
- 当高区已有 `2~3` 个强号时，应优先考虑这一层，而不是继续堆同区强分号。

## 5. 红球 `7` 码固定结构
下期双色球 `7红` 必须满足：
- `2~3` 个历史骨架号；
- `2` 个近邻斜连号；
- `1` 个多期斜连号；
- `1` 个即时跳点号；
- 若还有空位，再补 `1` 个结构平衡号或区带回摆号。

## 6. 红球硬性约束
最终 7 红建议优先满足：
- 低区（01~11）优先 `1~3` 个，常规主结构以 `2` 个左右为宜；
- 中区（12~22）优先 `2~3` 个，是最重要的承接区；
- 高区（23~33）优先 `1~3` 个；若上一期高区偏重，或最近连续两期是 `2:3:1 / 1:3:2` 一类中区强势结构，本期高区热号宜压到 `1~2` 个；
- 常规主结构优先考虑 `2:2:2`、`1:2:3`、`2:3:1`、`3:2:1`；
- 奇偶常规优先 `3:4` 或 `4:3`；若最近连续 `2` 期实际开奖已走出 `1:5 / 5:1` 一类极端偏态，多注时允许放行 `1` 注 `2:5 / 5:2` 的偏态结构；
- 不允许 4 个以上号码来自同一逻辑来源；
- 不允许原样照搬历史某一期 6 红组合。

### 6.1 准断区 / 断区识别与放行规则
定义：
- 准断区：低区、中区、高区三大区中，任意一个区只开出 `1` 个号码；
- 断区：任意一个区开出 `0` 个号码。

当满足以下任意 `2` 条时，视为"强准断区 / 断区信号"：
- 最近 `6` 期内，完整断区出现 `2` 次及以上，或准断区出现 `4` 次及以上；
- 上一期区间分布明显偏斜，如 `1:2:3`、`3:2:1`、`1:1:4`；
- 上一期高区或低区开出 `3` 个及以上，且当前候选池另一侧出现明显回摆号；
- 当前候选池中某一区缺乏强逻辑来源，很多号码只是为了凑均衡硬塞进去；
- 排序位明显偏移，如小号整体抬高、或中区密集而高区/低区同步走弱。

放行规则：
- 若只买 `1` 注，优先放行准断区结构，不默认强求 `2:2:2`；
- 若买 `2` 注及以上，至少保留 `1` 注准断区票；
- 完整断区只在信号很强时保留 `1` 注，不作为双色球默认主票结构；
- 不得因为票面"看着不均匀"就提前删掉合理的准断区 / 断区组合。

### 6.2 中区持续强势 / 高区单跳点补充规则
- 若最近连续 `2` 期都出现 `2:3:1`、`1:3:2` 或中区达到 `3` 个的结构，下一期默认中区仍为主承接区；
- 在这种情况下，前区 `7` 码中应优先放入 `2~3` 个中区号，且至少 `1` 个来自最近 1 期的邻号承接；
- 若高区只保留 `1` 个，优先从 `31~33` 一类边沿跳点或非最热高区号中选择，不优先继续堆 `28/29/31` 这类连续热高；
- 若高区主线已经连续 `2` 期未兑现，不得第三期继续机械追同一组高区热号。

### 6.3 重号集中 / 低区增强补充规则
- 若上一期红球中存在 `2~3` 个同时具备"重号延续 + 承接共振"的强候选，多注时必须保留 `1` 注重号集中票；
- 重号集中票的作用是集中吃主线，不得把所有强重号平均拆到不同票里导致"每注都沾边、没有一注打穿"；
- 若最近连续两期均为 `2:3:1`，则除中区强势票外，还必须保留 `1` 注低区增强票，优先结构为 `3:1:2`；
- 低区增强票应优先保留 `01/03/05/07/09/11` 一类低区活跃或承接口，不得因为前两期中区强就把低区统一压成 `1~2` 个。

### 6.4 均衡回摆 / 候选池防绑架补充规则
- 若最近两期分别走出 `2:3:1` 与 `3:1:2`（或反向镜像），下一期多注中必须保留 `1` 注 `2:2:2` 均衡回摆票；
- 均衡回摆票的作用是防止把上一期单次复盘结论直接放大成下一期主线，避免候选池前端整体偏斜；
- 当上一期高区刚由 `31/33` 一类边沿高跳兑现，下一期候选池必须同时复核 `28/29` 一类热高回切位；
- 当低区活跃位从 `01/03/07/11` 一类边口转为 `03/06/08` 一类平滑带，或进一步抬升到 `06/10` 一类偶低带时，候选池前端必须至少保留 `2` 个对应号码，不得继续只守旧低位边口。

### 6.5 高区续强回切 / 奇偶偏态放行补充规则
- 若上一期走出 `2:1:3` 且高区由 `24/25/32` 一类新高点主导，下一期多注时不得让"三高续强票"超过 `1` 注；至少保留 `1` 注 `2:3:1` 的高区单码回切票。
- 高区单码回切票优先从 `27/28/29` 一类热高回切位中取 `1` 个，不优先继续同时堆 `24/25/32` 三点。
- 若最近连续 `2` 期实际开奖都明显偶重或奇重，多注方案里必须保留 `1` 注偏态奇偶票，防止被常规均衡思维提前过滤。
- 偏态奇偶票的作用是承担极端结构，不得让所有票都固定在 `3:4 / 4:3` 的常规比值里。

## 7. 蓝球选号规则
### 7.1 单蓝规则
若只买 `1蓝`，必须同时参考：
- 最近 10 期蓝球活跃度；
- 近邻斜连 `±1`；
- 波动回补；
- 历史区间冷热平衡。

补充要求：
- 蓝球候选必须先列 `3` 个：`1个活跃号 + 1个斜连号 + 1个波动冷补号`；
- 若连续 `2` 期追同一个蓝球仍未中，下一期默认降为备选，不再直接单压。
- 若最近 `10` 期蓝球重复极少、呈明显分散态，则下调活跃号权重，上调深冷回补号权重；
- 深冷回补号一旦进入蓝球候选前三，实战单压时不得总是被活跃号机械挤掉。
- 若最近 `6` 期蓝球在同一小带内来回摆动（如 `02/06/07/08`），除邻号与冷补外，还必须显式保留 `1` 个 `3~6` 期内的活跃回访号；不得因为上一期刚开过某个小号，就把同带回访蓝整体降出前三。

### 7.2 禁止事项
- 禁止仅凭多期斜连就锁死 1 个蓝球；
- 禁止只看最近 1 期蓝球直接推下一期；
- 禁止因为某个蓝球在上一轮排第一，就持续追压。

## 8. 实际执行步骤
0. 采用"逐期自我修正"滚动口径：首次用第 `1~50` 期分析并预测第 `51` 期，开奖后立即复盘；然后用第 `2~51` 期预测第 `52` 期，依次滚动到最新一期。每一轮都必须执行"预测 -> 核对 -> 归因 -> 调参 -> 下一轮"闭环，不允许跳过复盘直接出下期票。
1. 统计最近 50 期高频对子、三组。
2. 先选出 2~3 个历史骨架号。
3. 再选 2 个近邻斜连号。
4. 选 1 个多期斜连代表号。
5. 强制加入 1 个即时跳点号。
6. 统计最近 `6` 期低中高三区分布，先判断是否存在准断区 / 断区高发与区带回摆信号。
7. 若最近连续两期中区偏强，优先把中区作为主承接区，再判断高区是否只保留 `1` 个边沿跳点号；但多注时必须额外保留 `1` 注低区增强票。
8. 若最近两期结构呈 `2:3:1` 与 `3:1:2` 的镜像摆动，多注时必须额外保留 `1` 注 `2:2:2` 均衡回摆票。
9. 若上一期存在 `2~3` 个强重号候选，多注时必须单独整理 `1` 注重号集中票，不能只把重号分散到各票对冲；同时检查候选池前端是否保留了 `2~3` 个均衡重组号。
10. 若上一期刚走出 `2:1:3` 的高区扩张，需额外整理 `1` 注 `2:3:1` 的高区单码回切票；边沿高跳刚兑现时，需显式复核 `28/29` 热高回切与 `06/08/10` 平滑低位带。
11. 若高区骨架偏重，或已触发回摆信号，再补 `1` 个结构平衡号或区带回摆号；不得让 `24/25/32` 一类新高点连续两期同时霸占候选池前端。
12. 若触发强准断区 / 断区信号，允许本期采用 `1:2:3`、`2:3:1` 等结构；多注时可保留 `1` 注完整断区票。
13. 检查最近 `2` 期奇偶比是否连续偶重或奇重；若是，多注时必须额外保留 `1` 注偏态奇偶票。区间与奇偶都是软约束，不是先验硬过滤。
14. 蓝球先列 `3` 个候选：`1活跃 + 1斜连 + 1波动冷补`；若蓝球处于分散期，需同时复核 `3~6` 期活跃回访号是否应升为最终单压，并优先确认红球主线是否已集中成票。
15. 若近3期蓝球集中在≤07的小号区，蓝球候选必须覆盖中段活跃带（08~12），至少1~2注分配给中段号（规则30）。
16. 若上一期高区归零或≤1个号，多注中应保留1~2注"高区单点"结构，防止弱回补导致的高区号过度预设（规则31）。
17. 检查上期号码的邻补候选（±1邻号），新邻补号应在至少2~3注中覆盖，不能仅作为边缘备选（规则32）。

## 9. 蓝球更稳的做法
若预算允许，比 `7+1` 更稳的是：
- `7+2` 或 `8+2`

因为双色球蓝球波动性明显高于红球，单蓝更容易出现"红球沾边、蓝球飞掉"的情况。

## 11. 复盘口径
每次开奖后必须回答：
1. 实际红球中有几个原本在模型前 15；
2. 没中的核心号，是候选池漏掉了，还是最终收口删错了；
3. 是否出现了区带回摆、准断区或断区结构，而我们仍被 `2:2:2` 的均衡思维限制；
4. 是否低估了中区持续强势，或把高区单跳点误判为高区整体扩张；
5. 是否把强重号主线拆散到了多注里，导致"各票沾边但没有集中票"；
6. 是否在连续两期 `2:3:1` 后，漏掉了应保留的低区增强票；
7. 是否把上一期复盘结论直接放大成下一期主线，导致候选池前端就偏了；
8. 在 `2:3:1` 与 `3:1:2` 交替后，是否漏掉了应保留的 `2:2:2` 均衡回摆票；
9. 高区是否只追了 `31/33` 一类边沿高跳，却漏掉了 `28/29` 热高回切；
10. 是否把上一期 `2:1:3` 误判为可以继续三高续强，漏掉了回到 `2:3:1` 且高区只留 `1` 码的回切结构；
11. 低区候选是否仍停留在 `01/03/07/11` 一类旧边口，或只做了 `03/06/08` 邻补，没有把 `06/10` 这类偶低上移位放进核心池；
12. 若最近连续两期都明显偶重/奇重，是否仍被常规奇偶均衡思维限制，没有保留偏态奇偶票；
13. 蓝球是否漏掉了 `3~6` 期内的活跃回访号；问题是模型错误，还是单蓝策略太死。
14. 是否在上期中区偏强（`2:3:1`）之后，完全没有设计高区突破票，导致"高区 `≥3` 个号"的结构被整组漏掉？
15. 前一期走出极端偶重后，是否仍然保留了偶重延续票，而没有替换为重奇票？若候选池中 `27/33` 这类边沿高跳的权重已降为 `0`，必须检查是否是因上期复盘结论的副作用导致的系统性漏号。
16. 是否漏掉了低区连号（`01/02` 或 `02/03`）的覆盖？低区被压空后，连号是最常见的回补路径，不能因边沿弱号就删掉。
17. 高区热号（近3期出现≥2次）是否被过度削减？高区热号一旦形成惯性，连庄概率高于反转概率。
18. 全偶/全奇特殊结构票是否只保留了1个低区号（低区支撑力不足）？全偶/全奇票的低区必须有2~3个低区号才能撑起结构。
19. 是否覆盖了本期新出现的中区连续对（如17/18）？热号（如27）是否被过度预设（>4注）？
20. 连续对是否被当作核心过度延续（规则24修正：连续对兑现后不应再作为核心构建）？低区是否以连号形式爆发而被漏掉？边沿高跳号是否被系统性清零？多热号同时衰减时是否集体降权？
21. 规则24修正后是否过度删除连续对中的单个号（如17）导致该号完全消失？低区是否连续两期爆发但覆盖仍不足（如08）？蓝球是否只追活跃带小号而忽略中段冷补？
22. 蓝球是否忽略了中段活跃带（08~12）的启动信号？是否只关注冷补号（09/11/13）而忽略了新活跃号（如10）？
23. 高区归零后的回补是否为"弱回补"（仅1个号）而非"强回补"（≥2个号）？多注是否全部预设了高区双点？
24. 新邻补号（如上期的邻号）是否在下一期快速扩散？是否应在至少2~3注中覆盖新邻补号？

## 20. 规则二十四再次修正（"消耗型信号"的边界）

**2040期首次触发即翻车后过度修正的问题**：2040期翻车后将`17`从8注全部移除（只保留在脱核重组票），但2041期实际开出`17`。

**最终修正**：连续对出现后的下一期，应将连续对拆开处理——其中一个号降为弱备选（最多2~3注），另一个号可正常参与候选池。不能因"连续对是消耗型信号"就把两个号一起从主线剔除。

## 21. 规则二十九（蓝球中段冷补路径）

蓝球策略长期聚焦活跃带小号（04/05/06/07），但近期多次开出中段号（09、11、13）。当小号活跃带连续出号≥3期后，中段冷补概率上升。

- 典型案例：2039蓝05→2040蓝04→2041蓝13（中段跳跃式回补）
- 判断标准：近3期蓝球集中在≤07的小号区。
- 应对策略：至少1~2注蓝球分配给中段号（09/11/13/15），不能只守活跃带内轮换。

## 22. 规则三十（蓝球中段活跃带启动信号）

当小号活跃带（≤07）连续出现≥3期后，中段号（08~12）可能以"新活跃号"形式出现（如2042期的`10`），而非"冷补号"。

- 典型案例：2039蓝05→2040蓝04→2041蓝13→2042蓝10（中段活跃带新启动）
- 判断标准：近3期蓝球集中在≤07的小号区，且中段号（08~12）开始松动。
- 应对策略：蓝球候选应覆盖中段全带（08/09/10/11/12），而非仅追冷补号（09/11/13）。至少1~2注分配给中段活跃带号码。

## 23. 规则三十一（高区弱回补识别）

高区归零（`3:3:0`或类似结构）后，回补可能是"弱回补"（仅1个号）而非"强回补"（≥2个号）。

- 典型案例：2041期`3:3:0`高区归零→2042期仅回补`31`一个号（弱回补），而非预期的2~3个号。
- 判断标准：上一期高区归零或高区≤1个号。
- 应对策略：多注方案应保留1~2注"高区单点"结构，而非全部预设"高区双点"。高区单点优先选择边沿跳点（31/32/33）或热高回切位（28/29）。

## 24. 规则三十二（邻补号扩散速度）

新出现的邻补号（如上期号码的±1邻号）在下一期的扩散速度可能快于预期。

- 典型案例：2041期`08`的低区邻补`07`、2041期`10/14`的中区邻补`12`，在2042期同时出现但预测覆盖不足（`07`仅1注、`12`仅1注）。
- 判断标准：上期号码存在明显的邻补候选（如`08→07/09`、`10→09/11`、`14→13/15`等）。
- 应对策略：新邻补号应在至少2~3注中覆盖，不能仅作为边缘备选。邻补号扩散票应与其他主线票并行配置。

## 25. 规则三十三（蓝球尾部活跃号识别）

蓝球中段活跃带（08~12）出号后，下一期可能出现"尾部活跃号"（13~16），而非继续在中段或回到小号区。

- 典型案例：2042期蓝球10→2043期蓝球16（从10跳到16，尾部活跃号兑现）。预测聚焦中段06~11，完全未覆盖16。
- 判断标准：上期蓝球为09~12的中段号。
- 应对策略：中段蓝球后，下期候选必须覆盖尾部号（13~16），至少1注分配给13/14/15/16，不能只守中段或小号。

## 26. 规则三十四（纯新号策略优化）

纯新号策略（完全不打重号）可避免热号惯性陷阱，但新号选择范围过广会导致分散。

- 典型案例：2043期纯新号4注，06/09/14/16/25/32都有覆盖但分散在不同注里，最佳仅2红。
- 问题根源：新号候选池过大（低区新号6个+中区新号8个+高区新号5个=19个），导致每注只能覆盖部分新号。
- 应对策略：①纯新号候选池收缩到10~12个最强新号；②明确1~2注"新号集中票"，把3~4个最强新号捆在一起；③邻补新号优先（如14是12/16的邻补、25是24的邻补）。

## 16. 规则二十四修正

**原规则24问题**：2040期首次触发即翻车——上期`17/18`连续对后，7注放入`17`或`17/18`，但实际全部消失。

**修正后规则24**：中区连续对属于"消耗型信号"，一旦兑现即失效。连续对出现的下一期，**最多只在1注中保留该连续对的其中一个号作为弱备选**，不得将其作为核心构建。

## 17. 规则二十六（低区连号爆发覆盖）

低区压空（≤1个号）后的下一期如果低区仍未爆发，那么**再下一期低区爆发概率急剧上升**，且往往以**连号形式**爆发。

- 典型案例：2037期低区仅1个号→2038期低区爆发`01/02`连号；2039期低区仅1个号(`08`)→2040期低区再次爆发`03/04`连号。
- 判断标准：连续两期低区≤2个号。
- 应对策略：至少在2注中设置低区双点（最好是连号或近距对，如`02/03`、`04/05`、`06/07`等），不得只放单点低区。

## 18. 规则二十七（边沿高跳号隔期回归）

边沿高跳号（`33/32/01/02`等极值位号码）的回归模式以"隔期回归"为主——即出现→消失1~2期→再次出现。

- 典型案例：`33`在2037期出现→2038期消失→2039期消失→2040期再次出现（典型的"消失2期回归"模式）。
- 判断标准：该号位于区间极端位置（01/02/32/33等），且近3期内出现过。
- 应对策略：每期候选池必须保留至少1注覆盖边沿高跳号，不能因"上一期没出就删掉"。对于`33`和`01/02`这类高频边沿号，建议在2注中分别覆盖。

## 19. 规则二十八（多热号集体降权）

当多个热号同时进入衰减期时（连续≥3期出现），整个热号组应集体降权，避免多注被同一批过气热号占据票面。

- 典型案例：2040期`25`(连续4期)、`27`(连续3期)、`30`(连续3期)三个热号同时衰减，全部零出现。预测中`25`仍入6注、`30`入5注，票面被过气热号严重占据。
- 判断标准：同一组内有≥2个热号同时进入第4期及以上。
- 应对策略：将所有进入衰减期的热号从"必覆盖"直接降到"可选可不选"，单热号覆盖率不超过3注；空出的票面空间分配给新信号（如低区连号爆发、中区新骨架等）。

## 12. 规则二十（高区热号惯性）

高区热号（近3期出现≥2次）形成惯性后，**下一期不默认收缩，而是继续保留1~2个该热号**。

- 典型案例：`27` 在 2037期（`27/29/31/33`）和 2038期（`27`连庄）形成惯性，规则20逆向判断"下期高区收缩"导致 27 被过度削减。
- 判断标准：近3期出现≥2次，即视为热号惯性。
- 应对策略：下一期必须在至少1注中保留该热号（建议放在高区惯性票中），而不是直接清零。

## 13. 规则二十一（低区压空后连号覆盖）

低区被压空（≤1个）后的下一期，**必须覆盖低区连号 `01/02` 或 `02/03`**，低区连号是压空后最常见的回补路径。

- 典型案例：2037期低区只有1个号（11），2038期走出连号 `01/02`，但候选池中 01 被标记为"弱候选"直接删掉。
- 判断标准：上期低区≤1个号，且候选池中 `01/02/03` 任一被清零。
- 应对策略：至少1注中放置 `01` 或 `02` 作为低区口，不应因"是边沿弱号"就直接删掉。

## 14. 规则二十四（中区连续对覆盖机制）

近3期未出现连续对（`±1`间距）时，下一期应主动在**中区**寻找连续对候选，不得只守固定骨架`13/15/22`而拒绝新信号。

- 典型案例：2039期走出`17/18`连续对（`17+1=18`），此前连续6期无连续对，但8注中无任何注覆盖17或18，中区骨架完全固化在13/15/22。
- 判断标准：近3期区间内任意两个号间距为1。
- 应对策略：连续对信号出现后，至少1注以连续对为核心构建（`17/18`、`16/17`、`18/19`等），连续对不得被强制拆分到不同注中。
- 规则25：热号惯性边际递减。热号连续出现3次后，下一期惯性概率下降，应将热号从"必覆盖"降级为"可选覆盖"，避免过度预设同一热号入超过4注。
- 典型案例：2039期`27`连续3期出现后，下一期零出现；`25`连续3期出现后，下一期降权保留但不再强制全注覆盖。

## 15. 一句话版本
下期双色球的新规则不是单纯把高分号往前排，而是：
- 先用历史重组定结构，
- 再用斜连补强，
- 最后强制留一个即时跳点，
- 蓝球则必须降斜连权重、提高波动补位权重。
