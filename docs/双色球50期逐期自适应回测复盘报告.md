# 双色球50期逐期自适应回测复盘报告

## 1. 回测设定
- 固定滚动窗口：`50` 期。
- 每期出票：`5` 注单式。
- 执行顺序：`1~50 -> 51`，`2~51 -> 52`，依次滚动至历史末尾。
- 每期开奖后立刻复盘，并在线更新红球权重、蓝球权重与票型优先级。

## 3. 最终学到的偏好
- 红球增强特征：`omission_mid(0.03), hot20(0.01), hot10(0.00), hot50(0.00)`
- 蓝球增强特征：`hot5(0.05), hot10(0.02), repeat_last(0.02), neighbor_last(0.00)`
- 高分票型：`主线票(3.49), 均衡票(3.49), 邻号票(3.49), 重号票(3.49), 冷补票(1.00)`
